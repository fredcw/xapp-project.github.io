XApp Project
Cross-Desktop Applications

A set of GTK based applications designed to be desktop and distro agnostic

Applications

Xed

xed is a small and lightweight text editor based on Pluma that supports most standard editing features, plus several not found in your average text editor (plugins being the most notable of these).

GitHub Repository

Xreader

Xreader (based on Atril) is a document viewer capable of displaying multiple and single page document formats like PDF and Postscript.

GitHub Repository

Pix

An image viewer and organizer based on gThumb with image editing features and advanced tools.

GitHub Repository

Xviewer

Xviewer is a simple image viewer, based on Eye of GNOME, which uses the gdk-pixbuf library. It can deal with large images, and zoom and scroll with constant memory usage. Its goals are simplicity and standards compliance.

GitHub Repository

Libraries

LibXApp

This project gathers the components which are common to multiple GTK desktop environments (Cinnamon, MATE and Xfce) and required to implement cross-DE solutions. It’s available in Python and JS as well, through GObject Introspection.

GitHub →

LibAdapta

libAdapta is a 'soft' fork of libAdwaita with themeing support. It provides the same features and the same look as libAdwaita by default but in desktop environments which provide theme selection, libAdapta apps follow the theme and use the proper window controls.

GitHub →

XApp Symbolic Icons

A set of symbolic icons which replaces the GNOME-specific Adwaita set.
